# ЭЗС — Streamlit дашборд обращений (v2)

Что есть:
- Фильтры: последняя неделя / выбор недели / диапазон дат + тематики + производитель
- KPI
- Динамика по неделям
- Разбивка по тематикам (в выбранном периоде)
- Производители × тематики (сколько обращений по каким проблемам) — в выбранном периоде
- Все обращения по месяцам за 2024–2025 (общая таблица) — по всем данным
- ТОП-5 ЭЗС по обращениям — как было

## Важно про доступ
В Google Sheets: Share → General access → Anyone with the link → Viewer

## Запуск
```bash
pip install -r requirements.txt
streamlit run app.py
```
