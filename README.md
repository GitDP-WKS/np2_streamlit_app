# ЭЗС — дашборд обращений (лёгкая версия, 2025)

Что делает:
- читает **сразу несколько вкладок** Google Sheets по списку GID (уже вшиты январь–декабрь 2025)
- строит таблицу **Причина (как в Google) × Завод (ЕПРОМ/НСП)**
- показывает **помесячную статистику 2025 (янв–дек)** по всем загруженным листам
- ТОП-5 ЭЗС (по фильтру)
- экспорт CSV/Excel

Важно:
- Google Sheets должен быть открыт: **Anyone with the link → Viewer**

Запуск локально:
```bash
pip install -r requirements.txt
streamlit run app.py
```
