# ЭЗС — Streamlit дашборд обращений

Дашборд читает данные напрямую из Google Sheets (CSV export) и строит:
- KPI (обращения, уникальные ЭЗС, топ-тематика, частая причина)
- динамику обращений по неделям (понедельник → воскресенье)
- разбивку по тематикам (правила по ключевым словам редактируются в сайдбаре)
- ТОП-5 ЭЗС по обращениям за выбранный период
- таблицу “сырых” данных после фильтров + выгрузка в CSV/Excel

## Важно про доступ
В Google Sheets: **Share → General access → Anyone with the link → Viewer**  
Иначе чтение по URL будет требовать авторизацию.

## Запуск локально
```bash
pip install -r requirements.txt
streamlit run app.py
```
