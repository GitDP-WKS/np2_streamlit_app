# ЭЗС — Streamlit дашборд (v5)

Что исправлено по твоему скрину:
- Разбивка теперь **НЕ по авто-тематикам**, а по **точным значениям “Причина обращения” как в Google Sheets**.
- Разбивка идёт **по заводам ЕПРОМ / НСП** (плюс “Другое/—”, если в данных встретятся другие значения).

Почему “по месяцам нули” раньше:
- Скрипт читал **только один лист (один GID)**.
- Если у тебя данные по месяцам лежат на **других вкладках**, нужно загрузить **все GID** и склеить.

Что сделано в v5:
- Поле “GID листов (через запятую)” — можно указать несколько GID, скрипт прочитает все и объединит.
- “Все обращения по месяцам (2024–2025)” считается по **всем загруженным листам**.
- Добавлен “Завод” через сопоставление текста производителя → ЕПРОМ/НСП (правится JSON-ом в сайдбаре).

Запуск:
```bash
pip install -r requirements.txt
streamlit run app.py
```
